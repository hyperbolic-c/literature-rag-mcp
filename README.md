# literature-rag-mcp

A Model Context Protocol server for literature RAG. Reads PDFs from Zotero and provides semantic search and Q&A capabilities.

## Installation

```bash
pip install literature-rag-mcp
```

## Usage

```bash
# Ingest documents
literature-rag ingest

# Check status
literature-rag status
```

## Configuration

Configuration is stored at `~/.config/literature-rag-mcp/config.json`.
